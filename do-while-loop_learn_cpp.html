Do While Loop in C++

The do-while loop is an exit-controlled loop. The loop body executes at least once, even if the condition is false initially.

Syntax:

do {
    // statements
} while(condition);

Flow of Execution:

  1. Execute the loop body first.
  2. Check the condition.
  3. If true, repeat step 1.
  4. If false, exit the loop.

Key Points:

  • Loop body executes at least once.
  • Condition is checked after execution.
  • Useful for menu-driven programs, input validation, games, etc.

Examples

1. Print 1 to 10

int i=1;
do {
    cout << i << " ";
    i++;
} while(i<=10);

2. Sum of First N Numbers

int n, sum=0, i=1;
cin >> n;
do {
    sum += i;
    i++;
} while(i<=n);
cout << "Sum = " << sum;

3. Factorial

int n, fact=1, i=1;
cin >> n;
do {
    fact *= i;
    i++;
} while(i<=n);
cout << "Factorial = " << fact;

4. Menu Driven Program

int choice;
do {
    cout << "1. Add\n2. Subtract\n3. Exit\n";
    cin >> choice;
    if(choice==1) cout << "Addition\n";
    else if(choice==2) cout << "Subtraction\n";
} while(choice!=3);

5. Number Guessing Game

int secret=7, guess;
do {
    cout << "Guess number: ";
    cin >> guess;
    if(guess!=secret) cout << "Wrong! Try again.\n";
} while(guess!=secret);
cout << "Correct!";

6. Print Digits of a Number

int n;
cin >> n;
do {
    cout << n % 10 << " ";
    n /= 10;
} while(n > 0);

7. Repeat Until User Enters 0

int num;
do {
    cin >> num;
    cout << "You entered: " << num << endl;
} while(num!=0);
