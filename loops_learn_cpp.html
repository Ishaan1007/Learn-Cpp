Loops in C++

A loop is used to execute a set of statements repeatedly until a certain condition is met.

Types of Loops:

  • for loop – used when number of iterations is known.
  • while loop – used when number of iterations is not known in advance.
  • do-while loop – executes at least once even if condition is false.

Loop Control Statements:

  • break – exits from loop immediately.
  • continue – skips current iteration and moves to next.

Nested Loops:

A loop inside another loop, commonly used for patterns and matrices.

Examples of Loops

1. Print numbers 1 to 10

for(int i=1; i<=10; i++) {
    cout << i << " ";
}

2. Factorial using while loop

int n, fact=1, i=1;
cin >> n;
while(i<=n) {
    fact *= i;
    i++;
}
cout << fact;

3. Multiplication Table

int n;
cin >> n;
for(int i=1; i<=10; i++) {
    cout << n << " x " << i << " = " << n*i << endl;
}

4. Fibonacci Series (do-while)

int n, a=0, b=1, c, i=1;
cin >> n;
do {
    cout << a << " ";
    c = a+b;
    a = b;
    b = c;
    i++;
} while(i<=n);

⭐ Pattern Examples

5. Right Triangle of Stars

for(int i=1; i<=5; i++) {
    for(int j=1; j<=i; j++) {
        cout << "* ";
    }
    cout << endl;
}

6. Inverted Triangle

for(int i=5; i>=1; i--) {
    for(int j=1; j<=i; j++) {
        cout << "* ";
    }
    cout << endl;
}

7. Pyramid Pattern

for(int i=1; i<=5; i++) {
    for(int j=1; j<=5-i; j++) cout << " ";
    for(int k=1; k<=2*i-1; k++) cout << "*";
    cout << endl;
}

8. Number Pyramid

for(int i=1; i<=5; i++) {
    for(int j=1; j<=i; j++) {
        cout << j << " ";
    }
    cout << endl;
}

9. Floyd’s Triangle

int num=1;
for(int i=1; i<=5; i++) {
    for(int j=1; j<=i; j++) {
        cout << num++ << " ";
    }
    cout << endl;
}

10. Hollow Square

for(int i=1; i<=5; i++) {
    for(int j=1; j<=5; j++) {
        if(i==1 || i==5 || j==1 || j==5) cout << "* ";
        else cout << "  ";
    }
    cout << endl;
}

11. Diamond Pattern

// upper pyramid
for(int i=1; i<=5; i++) {
    for(int j=1; j<=5-i; j++) cout << " ";
    for(int k=1; k<=2*i-1; k++) cout << "*";
    cout << endl;
}
// lower pyramid
for(int i=4; i>=1; i--) {
    for(int j=1; j<=5-i; j++) cout << " ";
    for(int k=1; k<=2*i-1; k++) cout << "*";
    cout << endl;
}
