Pointers in C++

A pointer is a variable that stores the memory address of another variable. Pointers allow direct access to memory and enable dynamic memory allocation, arrays, strings, and advanced concepts like data structures.

Key Concepts:

  • Declaration: int *ptr; (creates a pointer to int).
  • Address-of Operator (&): Gives memory address of a variable.
  • Dereference Operator (*): Accesses the value stored at the address.
  • Pointers are widely used in arrays, functions, and dynamic memory.

Examples of Pointers

1. Basic Pointer Usage

int x = 10;
int *p = &x;
cout << "x = " << x << endl;
cout << "Address of x = " << p << endl;
cout << "Value using pointer = " << *p;

2. Swapping Two Numbers

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

3. Pointer Arithmetic with Arrays

int arr[5] = {1,2,3,4,5};
int *p = arr;
for(int i=0;i<5;i++) {
    cout << *(p+i) << " ";
}

4. Find Maximum Element in Array

int arr[] = {10, 55, 23, 78, 44};
int *p = arr;
int max = *p;
for(int i=0;i<5;i++) {
    if(*(p+i) > max) max = *(p+i);
}
cout << "Maximum = " << max;

5. Pointer to Pointer

int x = 100;
int *p = &x;
int **pp = &p;
cout << "x = " << **pp;

6. Dynamic Memory Allocation

int *p = new int;
*p = 50;
cout << "Value = " << *p;
delete p;

7. String with Pointers

char str[] = "Hello";
char *p = str;
while(*p != '\0') {
    cout << *p;
    p++;
}

8. Function Returning Pointer

int* larger(int *a, int *b) {
    if(*a > *b) return a;
    else return b;
}

9. Pointer with Structures

struct Student { int id; char name[20]; };
Student s = {101, "Raj"};
Student *ptr = &s;
cout << "ID: " << ptr->id << " Name: " << ptr->name;
