Chapter 3: Control Structures

In C++, control structures allow us to alter the normal flow of execution of a program. They are divided into:

1. Decision Making

  • if: Executes a block if condition is true.
  • if-else: Executes one block if true, another if false.
  • else-if ladder: Checks multiple conditions sequentially.
  • switch-case: Multi-way branching based on a variableâ€™s value.

2. Looping

  • for loop: Used when the number of iterations is known.
  • while loop: Executes block repeatedly while condition is true.
  • do-while loop: Executes block at least once, then checks condition.

3. Jump Statements

  • break: Terminates the loop/switch immediately.
  • continue: Skips current iteration, goes to next.
  • goto: Transfers control to a labeled statement (rarely used).

Examples

1. if-else Example

#include <iostream>
using namespace std;
int main() {
    int age;
    cout << "Enter age: ";
    cin >> age;

    if(age >= 18)
        cout << "Eligible to vote";
    else
        cout << "Not eligible to vote";

    return 0;
}

2. for Loop Example

#include <iostream>
using namespace std;
int main() {
    for(int i=1; i<=5; i++) {
        cout << i << " ";
    }
    return 0;
}

3. switch-case Example

#include <iostream>
using namespace std;
int main() {
    int choice;
    cout << "1. Hello  2. Bye" << endl;
    cin >> choice;

    switch(choice) {
        case 1: cout << "Hello"; break;
        case 2: cout << "Bye"; break;
        default: cout << "Invalid choice";
    }
    return 0;
}
