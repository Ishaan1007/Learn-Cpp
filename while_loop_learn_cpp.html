While Loop in C++

The while loop is an entry-controlled loop. The condition is checked before executing the loop body. If the condition is true, the loop executes; otherwise, it stops.

Syntax:

while(condition) {
    // statements
}

Flow of Execution:

  1. Check the condition.
  2. If true, execute the loop body.
  3. Go back to step 1.
  4. If false, exit the loop.

Key Points:

  • Used when the number of iterations is not known in advance.
  • If condition is false initially, loop body never executes.
  • Be careful of infinite loops if condition never becomes false.

Examples

1. Print 1 to 10

int i=1;
while(i<=10) {
    cout << i << " ";
    i++;
}

2. Sum of First N Numbers

int n, sum=0, i=1;
cin >> n;
while(i<=n) {
    sum += i;
    i++;
}
cout << "Sum = " << sum;

3. Reverse a Number

int n, rev=0;
cin >> n;
while(n>0) {
    int digit = n%10;
    rev = rev*10 + digit;
    n /= 10;
}
cout << "Reverse = " << rev;

4. Factorial

int n, fact=1, i=1;
cin >> n;
while(i<=n) {
    fact *= i;
    i++;
}
cout << "Factorial = " << fact;

5. Palindrome Check

int n, rev=0, orig;
cin >> n;
orig = n;
while(n>0) {
    int d = n%10;
    rev = rev*10 + d;
    n /= 10;
}
if(rev==orig) cout << "Palindrome";
else cout << "Not Palindrome";

6. Count Digits

int n, count=0;
cin >> n;
while(n>0) {
    count++;
    n /= 10;
}
cout << "Digits = " << count;

7. Menu Driven Program

int choice;
do {
    cout << "1. Add\n2. Subtract\n3. Exit\n";
    cin >> choice;
    if(choice==1) cout << "Addition\n";
    else if(choice==2) cout << "Subtraction\n";
} while(choice!=3);
