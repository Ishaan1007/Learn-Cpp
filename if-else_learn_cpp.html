If-Else & Nested If (Advanced)

Control structures like if, if-else and nested-if allow decision-making in C++ programs. Instead of executing sequentially, they choose a path depending on conditions.

In exams, tough problems usually involve multiple conditions, mathematical checks, and nested blocks (if inside if).

Tough Examples in If / If-Else / Nested If

1. Largest of Three Numbers (Nested If) 
if(a > b) {
    if(a > c) cout << "Largest = " << a;
    else cout << "Largest = " << c;
} else {
    if(b > c) cout << "Largest = " << b;
    else cout << "Largest = " << c;
}
2. Type of Triangle 
if(a == b && b == c)
    cout << "Equilateral";
else if(a == b || b == c || a == c)
    cout << "Isosceles";
else
    cout << "Scalene";
3. Quadratic Equation Roots 
int d = b*b - 4*a*c;
if(d > 0) cout << "Real & Distinct";
else if(d == 0) cout << "Real & Equal";
else cout << "Imaginary Roots";
4. Leap Year (Nested If) 
if(year % 4 == 0) {
    if(year % 100 == 0) {
        if(year % 400 == 0) cout << "Leap Year";
        else cout << "Not Leap Year";
    } else cout << "Leap Year";
} else cout << "Not Leap Year";
5. Grading System 
if(marks >= 90) cout << "Grade A+";
else if(marks >= 75) cout << "Grade A";
else if(marks >= 60) cout << "Grade B";
else if(marks >= 45) cout << "Grade C";
else if(marks >= 33) cout << "Grade D";
else cout << "Fail";

Practice Questions (If-Else / Nested If)

  1. Write a program to find the greatest of four numbers using nested if.
  2. Check whether a character is a vowel or consonant (without logical operators).
  3. Develop a program to input marks of 5 subjects and print percentage & grade using if-else ladder.
  4. Write a program to check whether a person is eligible for driving license: - Age ≥ 18, must pass eye test (nested if).
  5. Check whether entered time is valid or not (hours: 0–23, minutes: 0–59, seconds: 0–59).
