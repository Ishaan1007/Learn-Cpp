Star Patterns in C++

Star patterns are common programming problems used to practice nested loops. They test your logic, control structures, and ability to handle spacing.

General Rules:

  • Outer loop = number of rows.
  • Inner loop = number of columns / stars / spaces.
  • Use spaces carefully to align the pattern.
  • Think row by row.

Tough Star Pattern Examples

1. Pyramid

*
***
*****
******* 
for(int i=1;i<=4;i++){
  for(int j=1;j<=4-i;j++) cout<" ";
  for(int k=1;k<=2*i-1;k++) cout<"*";
  cout<<endl;
}
