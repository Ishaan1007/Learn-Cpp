Chapter 2: Tokens, Expressions, and Data Types

Token: Smallest unit in a C++ program. Types of tokens:

  • Keywords: Predefined words like int, if, while.
  • Identifiers: Names given to variables, functions, etc. (e.g., marks, main).
  • Literals/Constants: Fixed values such as 10, 'A', 3.14.
  • Operators: Symbols for operations (+, -, *, /, ==).
  • Punctuators: Special characters ({ }, ;, ()).

Data Types:

  • Basic: int, char, float, double, bool.
  • Derived: Array, Pointer, Reference.
  • User-defined: Class, Structure, Enum, Typedef.

Expressions:

Combination of variables, constants, and operators that produces a value.

Example: a + b * c

Operators in C++:

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !
  • Assignment: =, +=, -=, *=, /=
  • Increment/Decrement: ++, --

Examples

1. Using Tokens

#include <iostream>
using namespace std;

int main() {
    int age = 20;      // age is identifier, 20 is literal
    cout << "Age: " << age;
    return 0;
}

2. Arithmetic Expression

#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 5, c;
    c = a + b * 2;   // Expression
    cout << "Result: " << c;
    return 0;
}

3. Using Operators

#include <iostream>
using namespace std;

int main() {
    int x = 10, y = 20;
    cout << (x < y);   // Relational (prints 1 = true)
    cout << (x == y);  // Equality (prints 0 = false)
    return 0;
}
